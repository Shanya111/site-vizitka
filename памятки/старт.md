# Старт проекта: Сайт-визитка Шаньяа

## Что это
Личный сайт-визитка. Показывает навыки автора и предлагает помощь тем,
кто хочет сделать себе сайт. Без пафоса — честно и по делу.

## Для кого
Для людей, которые перейдут на сайт и хотят узнать, что умеет автор.
Потенциальные клиенты на создание сайтов-визиток.

## Почему такие технологии
HTML + CSS + JS без фреймворков — оптимально для начала:
- Учишься понимать основу
- Легко деплоить (GitHub Pages / Netlify)
- Легко улучшать по частям

## Ключевые решения на старте
- Одностраничный сайт — быстрый старт, потом можно расширить
- Тёмная тема с фиолетовым акцентом (#7c3aed)
- Плавное появление секций при скролле через IntersectionObserver
